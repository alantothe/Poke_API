# Pokedex API Viewer

This project is a Pokemon Pokedex from the original 151 Kanto region. The app fetches data from the [PokéAPI](https://pokeapi.co/) and displays each Pokémon's name along with its sprite. Additionally, users can toggle between the regular and shiny sprites. Navigation through the Pokémon is made by using "Previous" and "Next" buttons.

## Features

- Fetches data from the PokéAPI
- Displays the name and sprite of each Pokémon
- Allows users to toggle between regular and shiny sprites
- Navigation through the Pokémon with "Previous" and "Next" buttons
